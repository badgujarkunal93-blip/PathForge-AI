# 🚀 PathForge-AI

> **Intelligent Path Navigation & Visualization Platform**

A cutting-edge web application that leverages AI to provide smart path planning, visualization, and analytics. Built with React, Three.js, and modern web technologies.

## ✨ Features

- **🤖 AI-Powered Path Planning** - Intelligent algorithms for optimal route optimization
- **📊 Data Visualization** - Interactive charts and 3D visualizations with Three.js
- **🎨 Smooth Animations** - Beautiful transitions and effects powered by Framer Motion and GSAP
- **📱 Responsive Design** - Fully responsive UI built with Tailwind CSS
- **⚡ Lightning Fast** - Built with Vite for optimal performance
- **🔥 Real-time Updates** - Firebase integration for dynamic data
- **📍 Interactive Maps** - Scroll-based animations with Lenis for smooth navigation
- **📈 Analytics Dashboard** - Visualize your path data with Recharts

## 📦 Installation

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn package manager

### Steps

1. **Clone the repository**
   ```bash
   git clone https://github.com/badgujarkunal93-blip/PathForge-AI.git
   cd PathForge-AI
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Set up environment variables**
   ```bash
   cp .env.example .env.local
   # Update .env.local with your Firebase credentials
   ```

4. **Run development server**
   ```bash
   npm run dev
   ```
   The application will be available at `http://localhost:5173`

## 🔐 Environment Variables

Create a `.env.local` file in the root directory:

```env
VITE_FIREBASE_API_KEY=your_api_key
VITE_FIREBASE_AUTH_DOMAIN=your_auth_domain
VITE_FIREBASE_PROJECT_ID=your_project_id
VITE_FIREBASE_STORAGE_BUCKET=your_storage_bucket
VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender_id
VITE_FIREBASE_APP_ID=your_app_id
```
